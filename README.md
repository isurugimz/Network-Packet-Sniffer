# Network Packet Sniffer & Analyzer
A Python-based tool to capture and analyze live network traffic.

## Features
- Captures Raw Ethernet Frames.
- Decodes IPv4 headers to extract Source/Destination IP.
- Identifies Network Protocols (TCP, UDP, ICMP).
